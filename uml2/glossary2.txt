WebBrowser - прикладное программное обеспечение для просмотра веб-страниц, содержания веб-документов, компьютерных файлов и их каталогов; 
управления веб-приложениями
Application -  программа, предназначенная для выполнения определённых задач и рассчитанная на непосредственное взаимодействие с пользователем
Authorization server - сервер выдающий токены доступа клиенту после успешной аутентификации владельца ресурса и получения разрешения
Content server - Компьютер, который хранит контент для Интернета. Контент (новости, спорт, ссылки и т. Д.) 
Отличается от данных транзакций, таких как записи и заказы клиентов
